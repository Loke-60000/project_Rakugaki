Research: Project Penguin Hive

A platform for posting text, images or drawings made directly on the site via an integrated application. Users will be able to create their own community and or talk via a private chat.

Inspiration

Miiverse by nintendo [2012-2017] Pixiv Sketch

Design

    Main color + mostly white (with dark mode and oled mode)

    Minimalist interface

    mascot appears to guide the user through certain sections such as the settings

Features

    Draw directly on the website

    Upload pictures

    Desktop and Mobile interface

    "Twitter like" account with profil pic and banner

    display name and username

    the possibility to create a community/group

    Private chat

    You can follow accounts

    Feed (following + random post)

    like and comment can be disabled depending on the user's taste

    Account deletion possible (account saved for 30 days in case of accidental deletion)

    themes (creating your own themes should be possible)

    Convert the web version into a downloadable application on smartphone if possible

    create a button on your page redirecting to one of your social networks (example: twitch channel)

    Possibility to notify your followers without making a post (example: "hey people I am live on youtube")
